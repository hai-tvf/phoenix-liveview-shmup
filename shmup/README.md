# Shmup (Phoenix LiveView)

Vertical scrolling shooter MVP: splash → playfield with mouse + hold-to-fire, server-side simulation, LiveView JS Hook for input and canvas rendering, `localStorage` high score.

## Docs

- Feature quickstart and toolchain: [`../specs/001-shmup-start-gameplay/quickstart.md`](../specs/001-shmup-start-gameplay/quickstart.md)
- Spec / plan / tasks: `../specs/001-shmup-start-gameplay/`

## Run

```bash
mix setup
mix phx.server
```

Open [http://localhost:4000](http://localhost:4000).

## Test

```bash
mix test
```
